Dashboard Help
About the Dashboard

The ECLI Metadata Dashboard provides a comprehensive view of the Portuguese judicial corpus. It allows you to explore document metadata, search for specific documents, and analyze the distribution of documents across courts and years.

Searching Documents

Use the search form to find documents based on specific criteria:

  • Court: Filter by court abbreviation (e.g., STJ, TRL)
  • Year: Filter by the year the document was published
  • Page Count: Filter by minimum and maximum number of pages

Click on any document in the search results to view its detailed metadata.

Understanding the Charts
  • Documents by Court: Shows the distribution of documents across different courts
  • Documents by Year: Shows the number of documents published each year
  • Document Metrics: Provides insights into document sizes and page counts

You can interact with the charts by hovering over data points, zooming in/out, and more.

Keyboard Shortcuts
  • Alt + S: Focus on search form
  • Alt + R: Refresh recent documents
  • Esc: Close any open modal
  • Alt + H: Open this help dialog
